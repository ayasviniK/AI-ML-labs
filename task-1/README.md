# Task 1 — Mobile Price Classification

Brief: Classification experiment to predict mobile price range using the provided dataset.

Contents
- `Mobile_Price_Classification-220531-204702.csv` — dataset used for training and evaluation.
- `task1.ipynb` — Jupyter notebook with data exploration, preprocessing, model training, and evaluation.

How to run
1. Open `task1.ipynb` in Jupyter Notebook / JupyterLab / Google Colab.
2. Ensure the dataset file is in the same folder (the notebook expects the CSV locally).

Recommended environment
- Python 3.8+
- Install dependencies:

```
pip install pandas scikit-learn matplotlib seaborn jupyter
```

Notes
- The notebook contains preprocessing steps, model training (classification), metrics, and visualizations.
- If you want to run the notebook end-to-end in a fresh environment, run the dependency install command first.
